## 动机
在我作为软件开发工程师的前5年职业生涯里,我一直专注于前端开发领域,游离于前端三大框架不亦乐乎,偶尔参与一些那个框架更好,那个更强大,
前端应该用Redux还是Mobx,或者状态新秀Zustand,又或者被后端伙计吐槽"你们前端不就是画页面的嘛!".<br/>
第6年的时候,我加入了一家外企,开始了我的全栈开发生涯.<br/>
当真正接触从前端到后端的软件开发全链路,一直到完成给用户的交付时,
我才体会到以前作为一个前端开发的短板,我似乎从来没有关心过应用数据的来龙去脉,也没有思考过当我的应用达到一定规模时我又该做出什么样的调整.<br/>
接下里我将会主要采用javaScript语言来一步步展示如何完成一个全栈项目,期待能为全栈学习的你们带来一些帮助.<br/>
准备好了吗? 让我们开始吧!

## 我们要做什么

<img src="https://s1.locimg.com/2024/08/25/7876c539016c5.png" alt="" title="" />
我们所要做的是一个可以记录某一次短途旅行行程的项目,我给它取名为Trip,我们将会包含以下功能<br/>
- 注册登录
- 填写行程信息(表单填写)
- 邮件抄送相关联系人行程详情
- exel文件导入行程信息
- 行程信息导出PDF和Exel文档
- 历史行程查询(表格操作)

## 在开始前你应该做什么
在开始学习你的全栈学习之旅前，你应该了解或做以下事情.<br/>
- 了解基本的 javascript 语法。
- 了解基本的 react（>= 16.8 版本）和 Nestjs 语法。
- 下载您喜欢的 IDE（推荐 VScode）
- 在本地安装 Nodejs。
- 在本地安装 Git。
