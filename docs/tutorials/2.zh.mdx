## 检查你的NodeJs
首先运行这个命令,确保你已经在本地下载了NodeJs,这是最重要的第一步! <br/>
```zsh
node -v
```
and
```zsh
npm -v
```
## 初始化一个新的react项目
其次,运行这个命令来创建你牛逼的react项目.
```zsh
npx create-react-app my-app --template typescript
```
我知道有人会说CRA已经过时了,我们应该使用CNA来初始化我们的react前端项目,但是,请打住!,
我们想要的是一个react项目,我们又为何要去初始化一个nextjs的项目呢,更何况,如果你使用过nextjs,
你就会发现,nextjs中存在很多和react不一致的开发习惯,以及一些破坏性的更改,那不是我们想要的,所以在我看来,CRA依然是
最有用的创建react项目的方式.
